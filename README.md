** Housing Price Prediction – Model Validation & Hyperparameter Tuning
 Project Overview**

This project focuses on predicting housing prices using Machine Learning while applying important model evaluation and optimization techniques. Rather than relying on a single model, multiple regression algorithms were trained and compared to understand their performance and reliability.

The project begins with data preprocessing and feature scaling, followed by splitting the dataset into training and testing sets. To ensure that the models perform well on unseen data, cross-validation was used for a more dependable evaluation. Overfitting was analyzed by comparing training and testing performance, helping to identify models that learned the training data too closely.

To further improve model performance, hyperparameter tuning was performed using GridSearchCV. This allowed the selection of the best model configuration based on validation results. The final models were evaluated using RMSE (Root Mean Squared Error) and R² Score, providing insights into prediction accuracy and overall model effectiveness.

Through this project, I gained practical experience in model validation, performance evaluation, overfitting control, cross-validation, and hyperparameter optimization. These techniques are essential for building reliable machine learning models that can generalize well to real-world data.

** Objectives**
Predict housing prices using regression models
Evaluate model performance using RMSE and R² Score
Identify overfitting and underfitting issues
Apply Cross-Validation for reliable model assessment
Optimize model performance using GridSearchCV
Compare multiple models and select the best-performing model
